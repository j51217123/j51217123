表格 Table 介紹

發表於 2021-12-20 分類於 HTML

在 div 布局還沒盛行之前，聽說都是用 table 進行排版，雖然現在用到的機會變少，但還是要對它有基礎的認識，避免書到用時方恨少，就替自己稍微筆記一下 !

在開始介紹之前，先附上簡易的 範例

  • 表格主結構
    • 分別使用 thead、tbody、tfoot 來定義表頭、表身、表尾

  • 表格元素

    • Caption 表格標題

      • 通常設定於 table 的第一個子元素，用於表示該 table 的 title

    • th 表格欄位標題

      • 針對表格中明確為欄位標題者，使用 th，例如下圖中的名稱、價格

    • tr 表格中的 row 元素

    • td 表格中每一個單元格，必需放在 tr 裡面

    • colgroup & col

      • span 可定義要選取的 column 數目

  • 常用表格 CSS 屬性

    • abbr：設定或取得該欄位的值，通常用於縮寫
    • colspan ：合併橫向儲存格
    • rowspan ：合併縱向儲存格
    • scope ：透過 scope 可以使閱讀器用更有效的方式朗讀，讓視障用戶根據標記一次性讀取一整列或行，通常設定 scope=“value”， value 可以是 col、row、colgroup、rowgroup
